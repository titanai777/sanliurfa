# 🌐 Çoklu Port Yapılandırması (1111, 1112, 1113)

## 📋 Port Tanımlamaları

| Port | Amaç | Mod | Açıklama |
|------|------|-----|----------|
| **1111** | Development | `dev` | Geliştirme modu, hot reload aktif |
| **1112** | Preview | `preview` | Build sonrası test, statik dosyalar |
| **1113** | Production | `production` | Canlı sunucu simülasyonu |

## ⚡ Hızlı Başlatma

### Windows'ta Çift Tıkla:
```
start-all.bat              ← Port seçici menü
start-1111-dev.bat         ← 1111 portu (Dev)
start-1112-preview.bat     ← 1112 portu (Preview)  
start-1113-prod.bat        ← 1113 portu (Production)
```

### NPM Script'leri:
```bash
# Development (1111)
npm run dev:1111

# Preview (1112)
npm run preview:1112

# Production (1113)
npm run preview:1113
```

## 🚀 Kullanım Senaryoları

### Senaryo 1: Geliştirme (1111)
```bash
# Kod yazarken, hot reload ile
npm run dev:1111
# http://localhost:1111
```

### Senaryo 2: Build Test (1112)
```bash
# Önce build al
npm run build

# Sonra preview et
npm run preview:1112
# http://localhost:1112
```

### Senaryo 3: Production Simülasyonu (1113)
```bash
# Production ortamında test
npm run build
npm run preview:1113
# http://localhost:1113
```

## 🔄 Aynı Anda Çoklu Port Kullanımı

Tüm portları aynı anda çalıştırabilirsiniz:

```bash
# Terminal 1 - Development
cd sanliurfa
npm run dev:1111

# Terminal 2 - Preview (başka terminal)
cd sanliurfa
npm run preview:1112

# Terminal 3 - Production (başka terminal)
cd sanliurfa
npm run preview:1113
```

**Sonuç:**
- http://localhost:1111 - Development
- http://localhost:1112 - Preview
- http://localhost:1113 - Production

## 📊 Port Kontrolü

### Windows:
```cmd
# Tum portlari kontrol et
netstat -ano | findstr "1111 1112 1113"

# Tek port kontrol
netstat -ano | findstr :1111
```

### Port Kapatma:
```cmd
# PID bul
netstat -ano | findstr :1111

# PID sonlandir (ornek: 1234)
taskkill /PID 1234 /F
```

## ⚠️ Hata Çözümleri

### "Port already in use" Hatası:
```bash
# Baska bir port dene
npm run dev:1112  # 1111 yerine 1112 kullan
```

### Tüm Portları Sıfırlama:
```cmd
# Tum Node.js sureclerini sonlandir
taskkill /F /IM node.exe
```

## 🔧 Port Değiştirme (Kalıcı)

`astro.config.mjs` dosyasını düzenle:
```javascript
const PORTS = {
  DEV: 3000,      // ← Yeni port
  PREVIEW: 3001,
  PROD: 3002
};
```
