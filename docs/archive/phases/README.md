- Operational phase governance files that are actively used by scripts remain in repo root:

  - `PHASE_CHANGELOG.md`
  - `PHASE_OPERATIONS_GUIDE.md`
- Historical per-range snapshot files were moved here to reduce root noise and improve navigation.
